<b>üë• User Roles & Responsibilities</b>
<p>The system defines three primary user roles with specific access levels:
</p>
<b>Admin</b>
<ul>
  <li>User Management: Create new users, customers, and staff accounts.</li>
  <li>Transaction Oversight: View all transactions.</li>
  <li>Sales Management: Manage product sales and generate receipts.</li>
  <li>Reporting & Records: Access customer reports and view all bill images.
</li>
</ul>
<b>Manager </b>
<ul>
  <li>Sales Tracking: View detailed sales information.</li>
  <li>Billing Status: Add and view pending bills.</li>
  <li>Performance Monitoring: See per-person reports including total sale amount, amount paid, and remaining amount due.</li>
  <li>Customer Records: View customer records linked to specific salesmen.</li>
  <li>Approval Workflow: Manage bills that require approval and those that do not.
</li>
</ul>
<b>Salesman / Labour</b>
<ul>
  <li>Receipt Generation: Take bill images and generate receipts for approved sales data.
</li>
</ul>
